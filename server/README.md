# Dating App Server

This is the backend for the Dating App using Node.js, Express, and MongoDB.

## Setup

1. Install dependencies:
   ```sh
   cd server
   npm install
   ```
2. Create a `.env` file (already provided) and set your MongoDB URI and JWT secret.
3. Start the server:
   ```sh
   npm run dev
   ```

The server will run on port 5000 by default.
